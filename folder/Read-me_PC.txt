                ALTER EGO + 1 CD-ROM


-------------------------------------------------------------------------------- 
       INTERACTIVE GLOSSARY
-------------------------------------------------------------------------------- 

This CD-ROM contains an interactive glossary that may be installed to a PC and 
enriched by its user. 

Adobe Air must be installed on your computer in order to use the glossary. 
Install Adobe Air via this link : http://get.adobe.com/fr/air/

The interactive glossary contents are located in the Ressources directory of the 
CD-ROM. Copy by going directly to the Lexique interactif file in the CD-ROM 
interface and click on the Ajouter un mot > Installer le lexique button.

-------------------------------------------------------------------------------- 
          AUDIO FILES
-------------------------------------------------------------------------------- 

Access all the audio MP3 files from the CD-ROM : go to the Enregistrements folder 
then click on Télécharger toutes les pistes).
It is not necessary to launch the CD-ROM to access the MP3 audio recordings. 
Access them in the ressources\mp3s directory.


-------------------------------------------------------------------------------- 
         LINUX USERS
-------------------------------------------------------------------------------- 

To launch this application, open the file AlterEgo+1.swf  (at the CD-ROM root 
directory) using a stand alone flash player.
